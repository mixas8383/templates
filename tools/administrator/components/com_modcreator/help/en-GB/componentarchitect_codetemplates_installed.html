Installed Code Templates Help

The following code templates will be pre-defined with the installation of Component Architect:

  • Joomla! 2.5 Standard
  • Joomla! 3.2 Standard
  • Joomla! 3.3 Standard

Do not change the records, plugins or files of a pre-defined Code Template!

For the most up-to-date information on the pre-defined code templates it is best to visit the Component Architect website (opens a new window).

Joomla! 2.5 Standard

The Joomla! 2.5 Standard code templates is based on the functionality in the core components of Joomla! 2.5 with code combining features from com_content, com_contact and com_banners.

Besides the folders, files and code for a component to be generated the Joomla! 2.5 Standard code template also provides other extensions to create a full package for a component.  This includes:

  • Modules for display of items
  • Search plugin
  • Smart Search plugin
  • Item Navigation plugin
  • Vote plugin
  • Pagebreak plugin

All these are selectable in the Component Definition and will only be generated into a new component when selected.

For the the full list of features supported by Joomla! 2.5 Standard code template see the Features Comparison tables (opens in new window).

Joomla! 3.2 Standard

The Standard Code Template for Joomla! 3.2 extends the functionality provided in the Joomla! 2.5 Standard code template by including the additional features in the Joomla! 3.x releases and meeting some revised class names and coding standards that were also introduced :

Additional functionality includes:

  • Support for tags
  • Support for versioning
  • Introduction of bootstrap
  • Move towards JQuery and redundancy of Mootools
  • evised administration layouts
  • Improved language support

For the the full list of features supported by Joomla! 3.2 Standard code template see the Features Comparison tables (opens in new window).
