Category New/Edit Help

This is where you can add a Component Architect Category or edit an existing Component Architect Category. Component Architect Categories allow you to organize records in the Component Architect component. Component Architect Categories are separate from other types of Categories, such as those for Articles, Banners, News Feeds, and so on.
