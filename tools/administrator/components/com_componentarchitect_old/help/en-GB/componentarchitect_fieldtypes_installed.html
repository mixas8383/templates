Installed Field Types Help

The following Field Types (listed alphabetically) are automatically installed with Component Architect Pro:

Access Level

This is the field type for a standard Joomla! Access field.

Calendar

This is the field type for a standard Joomla! Calendar field.

Category

This is the field type for a standard Joomla! Category field.

Checkbox

This is the field type for a standard Joomla! Checkbox field.

Checkboxes

This is the field type for a standard Joomla! Checkboxes field.

Color Picker

This is the field type for a standard Joomla! Color picker field.

Combo

This is the field type for a standard Joomla! Combo field.

Editor

This is the field type for a standard Joomla! Editor field. The editor field will be whatever editor is specified for the user.

Email Address

This is the field type for a standard Joomla! email address field.

File

This is the field type for a standard Joomla! File field.

File List

This is the field type for a standard Joomla! File List field.

Folder List

This is the field type for a standard Joomla! Folder List field.

Grouped List

This is the field type for a standard Joomla! Grouped List field.

Hidden

This is the field type for a standard Joomla! Hidden field. The attribute 'hidden' and class of 'hidden' is automatically used by Joomla! for a field of type 'hidden'

Image List

This is the field type for a standard Joomla! Image List field.

Integer

This is the field type for a standard Joomla! Integer field.

Language

This is the field type for a standard Joomla! Language field.

List

This is the field type for a standard Joomla! List field.

Media

This is the field type for a standard Joomla! Media field.

Modal

This is the field type for a standard Joomla! Modal field. Requires the generation of a modal dialog for a field. The dialog must be pre-coded.

Password

This is the field type for a standard Joomla! Password field.

Radio

This is the field type for a standard Joomla! Radio field.

Registry

This is a JSON Registry field type which will create a database field in which other Joomla! fields are stored. Each field that is to be stored in a Registry field must have the Registry field set in its Field record.

SQL

This is the field type for a standard Joomla! SQL Query field.

Tag

This is the field type for a standard Joomla! Tag field.

Telephone

This is the field type for a standard Joomla! Telephone Number field.

Text

This is the field type for a standard Joomla! Text field.

Text Area

This is the field type for a standard Joomla! Text Area field.

Time Zone

This is the field type for a standard Joomla! Time Zone field.

User

This is the field type for a standard Joomla! User field.

User Group

This is the field type for a standard Joomla! User Group field.

Web Address/URL

This is the field type for a standard Joomla! Web Address/URL field.
