Categories Manager Help

This view provides a list of the Component Architect Categories which the user can page through, filter and search. If authorised the user may also add, update, delete and reorder the Categories.

The Categories Manager is a standard component of Joomla!.

Filter Fields

Filter:

Enter text to search the list of Categories.

Search

Magnifying glass button to submit the Filter search.

Search Tools

Button to show/hide the filter fields.

Clear

Button to clear all values form the search and to reset the Filter fields.

Select Max Levels

A drop down list from which you can select a number of sub-category levels to display in teh list. A value can be selected and the list will then be filtered and redisplayed.

Select Status

A drop down list showing all Published/Status values in the list. A value can be selected and the list will then be filtered and redisplayed.

Select Access

A drop down list showing all Access values in the list. A value can be selected and the list will then be filtered and redisplayed.

Select Language

A drop down list showing all Language values in the list. A value can be selected and the list will then be filtered and redisplayed.

Select Tag

A drop down list showing all Tag values in the list. A value can be selected and the list will then be filtered and redisplayed.

Select Sort Field

Drop down list of columns in the view which can be selected as a sort column.

Select Sort Direction

Drop down list to select whether the records are to be sorted in ascending or descending order.

Number to Display

Drop down list to select the number of records to be displayed on each page.

Categories List

Check All

Selecting this box will select all Categories records on the page displayed.

Field Columns

The following fields are shown in the list for the Categories. The column headings may be clicked to change the sort field and whether it is ascending or descending sort.
  • Ordering - using drag and drop
  • Satus
  • Title
  • Access
  • Language
  • Id

Page Controls

When the number of items is more than one page, you will see a page control bar as shown below. The current page number being viewed is shaded.

  • Start: Click to go to the first page.
  • Prev: Click to go to the previous page.
  • Page numbers: Click to go to the desired page.
  • Next: Click to go to the next page.
  • End: Click to go to the last page.

Batch Processing

If choosing to copy a category, any other actions selected will be applied to the copied categories. Otherwise, all actions are applied to the selected categories.

Set Access Level

Select the access level that the records are to be set to or that will be set on the copied categories. On copy the default is to keep the orignal access level.
  • - Keep orginal Access levels -
  • Public
  • Registered
  • Special

Set Tags

Select the tags that the records are to be set to or that will be set on the copied categories. On copy the default is to keep the orignal access level.

Set Language

Select the language that the records are to be set to or that will be set on the copied categories. On copy the default is to keep the orignal language.
  • - Keep orginal Language -
  • All
  • English (UK)
  • Any other installed language

Select Category

Select the category that the records are to be set to or that will be set on the copied categories. On copy the default is to keep the orignal category.

Select Copy

Select whther to move or copy the selected items. If copy is selected any changes will be applied to the copied items.

Process

Button to process the batch options set for the selected items.

Clear

Button to clear the batch options..
